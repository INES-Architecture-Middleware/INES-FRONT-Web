#### Project Setup

1. Create `.env` file:

```bash
touch .env
```

2. Add the VITE_URL_GATEWAY variable in .env with your API Gateway URL:

```
VITE_URL_GATEWAY=your_api_gateway_url
```

3. Install dependencies and start the development server:

```bash
npm install
npm run dev
```
